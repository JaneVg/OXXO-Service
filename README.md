# OXXO-Service
OXXO Service es un software desarrollado en Python que simula el funcionamiento de un cajero automático inteligente especializado en tiendas OXXO.
El sistema permite:
Consultar facturas de servicios básicos (agua, luz, predio).
Generar recibos digitales con folio único.
Realizar pagos ficticios con confirmación.
Guardar comprobantes en archivos de texto.
Interactuar mediante una interfaz gráfica simple (Tkinter).
Este proyecto busca demostrar cómo un software especializado puede automatizar operaciones financieras básicas, mejorar la experiencia del usuario y servir como base para futuras integraciones más complejas.


Requisitos
Python 
Librerías necesarias (se instalan con pip):
pip install tk
Un editor de texto o IDE (VS Code, PyCharm o incluso IDLE de Python).

Instalación
Clonar o descargar el repositorio/código del proyecto.
git clone https://github.com/JaneVg/OXXO-Service
cd OXXOService


Guardar el archivo principal.
Comandos de ejecución
En consola, ejecutar:
python oxxo_service.py
Esto abrirá la ventana del cajero OXXO Service.

Ejemplo de uso
Seleccionar un servicio en el menú desplegable (ejemplo: Agua).
Ingresar el código del servicio (ejemplo: AG1001).
Presionar Consultar factura → aparecerá la información del titular, monto y folio.
Presionar Pagar factura → confirmará el pago ficticio y mostrará la fecha de pago.
Presionar Guardar recibo → generará un archivo .txt con el comprobante.
Presionar Limpiar → reinicia el formulario para otra operación.
